PhotoGenic is an AIR desktop application created for Photography enthusiasts of all ages. The world-famous photo hosting web site, Flickr, consists of a vast archive of millions of images that are uploaded by Flickr users from all around the world. PhotoGenic aims to enable any user, not just registered Flickr users, to be able to browse the various categories of images hosted on the Flickr web servers without requiring the use of a web browser.

To use PhotoGenic, simply enter a tag in the search field and images are loaded from the Flickr web servers. Browse through page after page of image thumbnails. If you see an image you like and want to take a closer look, click on a the thumbnail. Your web browser will open a larger version of the image. It's that easy!
